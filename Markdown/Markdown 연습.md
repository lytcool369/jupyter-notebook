# 공식

$$
C = \frac{1}{2} \sum_{n=1}^{n}(t_{n} - y)^2
$$

<br>

$$
C = \frac{1}{2} \sum_{n=1}^{n}\lVert (t_{n} - y)_2^2 \lVert
$$

문장 사이에 [식] $C = \frac{1}{2} \sum_{n=1}^{n}\lVert (t_{n} - y)_2^2 \lVert$ 을 넣어준다.
